# CSC369 — Lower-Level Systems (Fall 2025)

This repository contains my completed assignments for **CSC369: Lower-Level Systems** at the University of Toronto (Fall 2025).  
All projects are implemented in **C** and focus on core operating system concepts.

---

## Topics & Assignments

### Concurrency
**Folders:**
- `Cooperative Threads`
- `Preemptive Threads`
- `Queue Data Structure`

Implemented:
- Locks and condition variables
- Thread-safe queues
- Cooperative and preemptive scheduling
- Correct synchronization and race-condition handling

---

### Virtual Memory
**Folder:**
- `Virtual Memory`

Implemented:
- Page tables and address translation
- TLB behavior
- Page fault handling

---

### File Systems (FUSE)
**Folders:**
- `FUSE File Systems`
- `Fuse Passthrough`

Built a user-space file system using **FUSE**, including:
- Inodes and directories
- Path resolution
- POSIX file operations (open, read, write, stat, etc.)

---

### Utilities
**Folder:**
- `Trace File Generator`

Tools for generating and analyzing execution traces for testing and validation.

---

## Tools & Environment

- Language: **C**
- Platform: **Linux**
- APIs: **POSIX**, **FUSE**
